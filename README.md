# 🛒 E-Commerce App

A modern e-commerce mobile application built using Flutter, designed to provide users with a smooth and intuitive shopping experience. The app allows users to browse products, manage their cart, and interact with a clean and scalable system architecture.

---

## 📌 Problem Statement

Online shopping applications often suffer from complex user interfaces and unstructured workflows, making it difficult for users to browse products and manage purchases efficiently.

---

## 💡 Solution

This application provides a clean, user-friendly interface combined with a scalable architecture to simplify the shopping experience. It focuses on usability, performance, and maintainability.

---

## ✨ Key Features

* 🔐 User Authentication (Login / Register)
* 🛍️ Product Listing & Browsing
* 📦 Product Details View
* 🛒 Add to Cart functionality
* ❤️ Wishlist / Favorites
* 📱 Clean and responsive UI
* ⚡ Fast and smooth user experience

---

## 🧱 Architecture

The project follows **Clean Architecture** principles to ensure scalability and maintainability.

```
lib/
├── core/
├── features/
│   ├── auth/
│   ├── home/
│   ├── cart/
│   └── product/
```

## 🎨 Design System

The app is built with a structured design system including:

* Typography System (Reusable Text Styles)
